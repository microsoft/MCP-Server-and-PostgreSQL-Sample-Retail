## 安全性

微软非常重视我们软件产品和服务的安全性，这包括我们 GitHub 组织中的所有源代码库。

**请不要通过公开的 GitHub 问题报告安全漏洞。**

有关安全报告的信息、位置、联系方式和政策，请查看微软代码库的最新指南：
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)。
