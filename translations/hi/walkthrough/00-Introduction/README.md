### Row Level Security (RLS)

PostgreSQL RLS рдбреЗрдЯрд╛ рдЖрдЗрд╕реЛрд▓реЗрд╢рди рд╕реБрдирд┐рд╢реНрдЪрд┐рдд рдХрд░рддрд╛ рд╣реИ:

  FOR ALL TO regional_managers
  USING (store_id = ANY(get_user_store_list()));
